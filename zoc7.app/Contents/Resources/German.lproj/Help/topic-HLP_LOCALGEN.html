Local Shell Überblick

Der Verbindungstyp Local Shell baut eine Pseudo-Verbindung zur Shell des lokalen Rechners aus. Auf dieser Weise kann über ZOC auf das lokale Terminal zugegriffen werden, z.B. als Alternative zum Mac OS X Terminal. 

Da ZOC grundsätzlich verbindungsbezogen arbeitet, muss beim Verwenden der Local Shell eine Zieladresse für die Verbindung angegeben werden. Gültige Adressen sind entweder localhost oder 127.0.0.1 oder der Name jedes beliebigen Systembefehls in Shebang Format, z.B. #!/usr/bin/login -p harry, #!/usr/bin/ksh oder #!/usr/bin/ssh secure.hogwards.edu (nur wenige Systembefehle machen an dieser Stelle wirklich Sinn, zumindest aber sollten die Befehle interaktiv sein). 

Falls kein Systembefehl übergeben wurde, führt die Local Shell den Befehl aus, der unter Local Shell Einstellungen eingetragen wurde (die Voreinstellung ist /usr/bin/login).
