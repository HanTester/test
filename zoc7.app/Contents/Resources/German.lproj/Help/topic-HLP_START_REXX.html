Autostart REXX-Programme

Zwei REXX-Programme werden beim Starten von ZOC automatisch ausgeführt. Das erste Programm, ZOCEVENT.ZRX (welches auch beim Schließen von ZOC noch einmal ausgeführt wird), kann dazu verwendet werden, um Fax-Programme (wie z.B. 'FaxWorks') aus- bzw. wieder einzuschalten. Die Datei ZOCEVENT.TXT im REXX Ordner enthält hierzu weitere Informationen.  

Das zweite Programm heißt STARTUP.ZRX und wird erst nach dem Initialisieren von ZOC ausgeführt. Es kann dazu verwendet werden, um z.B. bei jedem Starten von ZOC eine bestimmt Gegenstelle anzuwählen, eine Begrüßung anzuzeigen o.ä.
