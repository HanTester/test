Verknüpfung

       
Diesen Adressbucheintrag auch im Datei-Menü des Hauptfensters anzeigen
  Das Anwählen dieser Option erzeugt einen Eintrag im Datei-Menü von ZOC, so dass der Adressbucheintrag von dort aus schnell angewählt werden kann.
 
Verbindung per F-Taste...
  Hier kann eine Funktionstaste festgelegt werden, über die (zusammen mit der Strg-Taste, z.B. Strg+F6) eine Verbindung zu der Gegenstelle aufgebaut werden kann.
 
Verknüpfung auf der Arbeitsoberfläche erstellen...
  Dieser Knopf erstellt ein Icon mit dem Aufruf von ZOC und gleichzeitigem Verbindungsaufbau zu dieser Gegenstelle auf der Arbeitsoberfläche.
