ZOC Versionen

 

ZOC ist ein Modem und Telnet-Kommunikationsprogramm, das in verschiedenen Ausführungen unter den Betriebssystemen Windows und Mac OS X verfügbar ist.  

Der Menüpunkt 'Produktinformation' im Hilfemenü gibt Aufschluss über die gerade verwendete ZOC-Version.

       
ZOC im Vorabtest bzw. Vollversion mit Lizenzcode
  Das ist die Demoversion von ZOC, die 30 Tage verwendet werden darf. Es sind keine Funktionen ausgeschaltet oder weggelassen, nach jedem Download wird aber das Bestellformular geöffnet, zur Erinnerung an die Tatsache, dass nach 30 Tagen das Programm entweder gekauft oder gelöscht werden muss.
Wenn Sie ZOC kaufen, erhalten Sie einen Lizenzcode sowie eine Anleitung, wie der Vorabtestmodus abgeschaltet wird (und damit auch die Erinnerungsfenster der Vorabtestversion).
Nach Eingabe eines Lizenzcodes werden die Erinnerungsfenster und Meldungen des Vorabtests deaktiviert. Damit handelt es sich um die uneingeschränkte Vollversion. Sie können jederzeit von unserer Webseite eine neue Testversion downloaden und mit dem Lizenzcode in die Vollversion verwandeln, so lang die Versionsnummer zum Lizenzcode paßt (z.B. Download von Version 6.28 mit einer 6.xx Lizenz).
 
Versionspolitik
  Nach der V6.00 wird es eine Folge von ZOC Versionen geben, die Fehlerbereinigungen und kleinere Verbesserungen (wie z.B. einige neue Optionen, verbesserte Dialoge o.Ä.) enthalten. Dies wird so lange fortgesetzt, bis ein Punkt erreicht ist, an dem ZOC verhältnismäßig stabil und komfortabel läuft. All diese Versionen werden die gleiche Hauptversionsnummer haben, d.h. sie werden durch die 6.xx Lizenz abgedeckt sein. Dieser Punkt wird wahrscheinlich im Bereich um die Versionnummer 6.40 herum erreicht sein.
Danach wird es eventuell eine neue Version mit deutlichen Verbesserungen geben. Diese Version würde dann V7.00 heißen. Benutzer mit einer 6.xx Lizenz könnten dann eine 7.xx Lizenz zu einem reduzierten Preis erwerben (momentan kostet z.B. das Update von 5.xx auf 6.xx ca. 29 Euro).
 
Platformen (Windows bzw. Mac OS X)
  ZOC ist für Windows (ab Windows 2000) und Mac OS X (ab OS X 10.4 Tiger) verfügbar. Soweit es die Betriebssysteme zulassen sind die Funktionen des von ZOC unter den entsprechenden Platformen identisch. Konfigurationsdateien können zwischen den Platformen ausgetauscht werden und die Lizenz kann entweder für Windows oder OS X verwendet werden.
