Unix Sockets Overview

The unix sockets lets you access applications that provide a character based data stream via a unix domain socket on your local machine. This means that you can use ZOC to communicate with another application if the application provides a unix socket for communication. For example, some virtual machine applications provide unix sockets as replacements for physical serial terminal ports. 

These sockets are represented by a data point in the local file system; hence they look like a filename. To connect to such a socket, use ZOC's Quick Connect function or create a host directory entry and provide the name of the socket in the Connect to field, e.g. /tmp/server/somesocket.
