REXX ZocDeviceControl

Available commands for the ZocDeviceControl script command (see ZOC-REXX Commands):  

       
GETRS232SIGNALS
  Return a string that indicates the raised incoming signals (CTS, DSR, CD, RI) on the serial port, e.g. ##[CTS][DSR][CD]##
Note: For backward compatibility the command GETSTATE returns the same result as GETRS232SIGNALS).
 
TESTACCESS <name>
  This command tests if the serial port <name> (e.g. TESTACCESS COM3 or TESTACCESS /dev/cu.pl2303) is valid and accessible.
