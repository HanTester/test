REXX ZocDeviceControl

Available commands for the ZocDeviceControl script command (see ZOC-REXX Commands):  

       
RESOLV <hostname or ip>
  Returns hostname and ip for the requested host in the form "<primary hostname>; <ip>" (e.g. RESOLV mail.emtec.com might return www2.emtec.com; 212.34.172.147).
