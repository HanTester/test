VT102/VT220/Linux/Xterm Keys on the PC Keyboard

These keys can be mapped to user buttons (Options→Session Profile→User Buttons) or to any keyboard keys in the Options→Keyboard Profiles by using Emulation Dependent Key Names for Remapping They can also be sent this way from Rexx through the ZocSetEmuKey command.
