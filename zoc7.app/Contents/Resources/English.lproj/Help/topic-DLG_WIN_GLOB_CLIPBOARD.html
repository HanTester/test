Clipboard

       
Automatically copy selected text to clipboard
  When you select text in the terminal area, with this option is enabled, the program automatically copies the selected text to the clipboard and clears the selection.
 
Suppress 'Text copied to clipboard' message
  This option can be used to suppress the small confirmation window that ZOC usually displays when copying text to the clipboard.
 
Use Ctrl+V and Ctrl+C for clipboard
  In terminal software, Ctrl+V and Ctrl+C are key combinations which send specific codes to the remote computer (e.g. Ctrl+C is usually sending a hex 03 byte which some system use to interrupt an ongoing procedure). If you do not need these functions, you can set this option to use Ctrl+V to paste text from the clipboard into the session and Ctrl+C to copy selected text to the clipboard.

Note: Ctrl+C is only affected while text selected on screen. With no selected text, Ctrl+C will send hex 03, which is mostly interpreted as a 'break'.
 
Paste without final 'newline'
  Pasting text with this option checked, will keep the cursor in the line where the pasted or quoted text ends (otherwise it will also paste a final Enter key).
 
Immediately paste single-line selections
  Selecting this option will always send marked text immediately, if only one line was marked on the screen. If the option is half-selected, the text can still be sent immediately if you hold the Shift key pressed while doing a single-line selection.
 
Indent quoting
  If this option is checked, quoted texts will be indented by one space (e.g. to prevent mail programs from reformatting the text). This option is ignored when quoting in block-style.
 
Always reformat quotings
  If this option is activated, the Send-Quoted function from the Edit Menu will always try to reformat the text before sending it. When disabled, reformatting is still available by holding the shift key pressed.
 
Width of quoted pasting
  Reformatting for smart quoting will be done in a way that no line is longer than the number of characters specified here.
