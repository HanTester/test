REXX ZocDeviceControl

Available commands for the ZocDeviceControl script command (see ZOC-REXX Commands):  

       
RESOLV <hostname or ip>
  Returns hostname and ip for the requested host in the form "<primary hostname>; <ip>" (e.g. ZocDeviceControl("RESOLV mail.emtec.com") may return www2.emtec.com; 212.34.172.147).
 
TCP_NODELAY ON
  Enables the TCP_NODELAY option on socket level (google for TCP_NODELAY or Nagle's algorithm), e.g. CALL ZocDeviceControl "TCP_NODELAY ON"
 
GLOBAL noreverselookup
  Sets the global telnet option to not do DNS reverse lookups on the destination address.
 
GLOBAL reverselookup
  Sets the global telnet option to do DNS reverse lookups on the destination address.
 
GLOBAL noping
  Sets the global telnet option to not ping a host before making the connection.
 
GLOBAL ping
  Sets the global telnet option to ping the host before connecting to the destination address.
