About the Evaluation Version of ZOC

 

General

ZOC is a commercial software product which you can try before you buy it. The trial is fully functional and you can use it for 30 days (in fact on 30 different days) to find out if you like the program and if its features meet your requirements.  

After that time you should either order a license or uninstall the program from your system.  

 

 

Evaluation Mode

When the program displays the text evaluation mode in the window title bar, it means that there is no licence code from a ZOC purchase on your computer.  

This will obviously be the case when you are evaluating the software for the first time. It can also happen if someone installed the program on your computer and did not tell you that you need to purchase a license (amazingly this happens quite often). 

If you did purchase ZOC, but see ZOC reporting evaluation mode, maybe you did not enter the license code which you received with the purchase. Or you reinstalled your system without reentering the code or you have a license code for an older version of ZOC. 

If you are not sure why you are seeing the evaluation message, please do not hesitate to contact our support.  

 

 

Restrictions in Evaluation Mode

There are no technical restrictions in evaluation mode, i.e. the evaluation version of ZOC is fully functional. We did not disable any functions and everything that has to do with the evaluation period is clearly marked as such (for example you will see extra windows to remind you of the programs' evaluation status).  

If something does not work the way you expect it, it is either by design, by misconfiguration or it may be caused by a program bug. In any case, you can contact our support during the evaluation period and we will do our best to solve the problem.
