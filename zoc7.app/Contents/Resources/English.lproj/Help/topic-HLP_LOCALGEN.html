Local Shell Overview

The local shell communication type will create a pseudo connection which opens a shell on the local computer. This way you can open local terminal sessions within ZOC, thus replacing the Mac OS X terminal. 

Since ZOC is essentially connection related, when using Local Shell you need to specifiy a destination to connect to. Valid destinations are either localhost or 127.0.0.1 or the name of any system command in shebang format, e.g. #!/usr/bin/login -p harry, #!/usr/bin/ksh or #!/usr/bin/ssh secure.hogwarts.edu (only few system commands actually will make sense here, they should at least be interactive). 

If not being passed a system command, the Local Shell handler will run the command from the Local Shell Settings which by default is /usr/bin/login.
