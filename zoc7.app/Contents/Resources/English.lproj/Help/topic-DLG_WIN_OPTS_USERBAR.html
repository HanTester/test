User Button Bar

Below the tool bar resides a group of user defined buttons that can be managed here.  

 

User Buttons

These buttons, when clicked, can perform a variety of functions, among them the sending of text or commands, the simulation of a ZOC menu command or the execution of a REXX script. This part of the session profile allows you to add, modify or delete user buttons. 

You will find more information about the functions that can be mapped onto these buttons, by editing an item and then clicking Help in the item dialog (see Action Assistant).
