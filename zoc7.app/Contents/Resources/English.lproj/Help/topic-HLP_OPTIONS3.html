Program Settings

Program options (as opposed to the options from the Session Profiles) are valid for all connections. They contain options which, once they are set, hardly ever change. 

Program options are

  • Window
  • Tabs
  • Printer
  • Folders
  • Scrollback
  • Clipboard
  • Mouse
  • Sounds
  • Disconnect Action
  • Passwords
  • AutoConnect
  • Special Files
  • Subsystems
  • Prompts
  • Miscellaneous
